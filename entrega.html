Entregas

Aqui na Mato Verde, nos preocupamos em entregar seus itens com segurança e rapidez. Por isso, oferecemos diferentes opções de entrega, para atender às suas necessidades.

Entrega via Correios (Kangu)

Para itens de até uma caixa, oferecemos a opção de entrega via Correios (Kangu). Essa opção é mais econômica e adequada para itens pequenos e leves.

Entrega via transportadoras

Para itens com valor superior a uma caixa ou saca, oferecemos a opção de entrega via transportadoras. Essa opção é mais segura e rápida, e é adequada para itens maiores e mais pesados.

Cotação automática

Se o cliente não informar a transportadora de sua preferência, a Mato Verde fará uma cotação automática das melhores transportadoras para o seu pedido. A transportadora mais barata será escolhida automaticamente.
